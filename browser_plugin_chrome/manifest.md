permissões:

storage: 
    - página: https://developer.chrome.com/docs/extensions/reference/api/storage
    - função: guardar dados do usuário mesmo após ele fechar o plugin
    - onde acho que vai ser útil: Retornar a pesquisa caso o plugin seja fechado
    antes da pesquisa realizada (exemplo clicar na lupa)

activeTab:
    - página: https://developer.chrome.com/docs/extensions/develop/concepts/activeTab
    - função: permite controle da página atual quando o usuário clica no plugin
    - onde acho que vai ser útil: Acessar o conteúdo da barra de pesquisa

Icons:
Os icones do botão da barra de pesquisa, em 3 tamanhos (talvez seja possível deixar só um)
